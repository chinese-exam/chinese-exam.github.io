# 模拟考试系统 - 开发者文档 (v0.3.1)

## 📅 项目现状 (截至 2026年5月14日)

当前项目已演进为一个**完全递归、支持多级分类**的云端测评平台。v0.3.1 优化了题库组织标准，采用更简洁的纯递归映射结构。

### 🚀 近期主要更新 (v0.3.1)
- **精简递归架构**：`library.json` 升级为纯递归映射，移除了冗余的 `children` 键。
- **动态路径解析**：自动解析试卷在树中的位置，支持多级面包屑展示。
- **服务层优化**：`exam-service` 的 `flattenLibrary` 工具已适配纯映射结构，支持无限层级遍历。
- **智能搜索**：支持对试卷名称及所属各级分类路径进行全文本搜索。

## 🎯 核心功能

✅ **无限级分类管理** - 支持任意深度的题库嵌套，无需额外关键字  
✅ **全流程考试引擎** - 考前确认、实时计时、进度存盘、自动评分、公式渲染  
✅ **动态远程导入** - 支持 GitHub Repo 或任意 CDN 上的 JSON 试卷实时加载  
✅ **Zod 运行时校验** - 严格校验递归映射结构与试卷数据  

## 📋 题库标准 (Library Standard v0.3.1)

`library.json` 采用纯递归的键值对结构：

```json
{
  "官方题库": {
    "初级": {
      "语文": "/questions/chinese_easy.json"
    },
    "数学": "/questions/math_easy.json"
  },
  "外部考试": "https://example.com/other.json"
}
```

- **键**：分类名称或试卷名称。
- **值**：字符串（试卷 URL）或对象（下一级分类映射）。

## 🚀 开发者工具

### flattenLibrary
用于将树状映射转换为扁平列表，每个元素包含 `path` 数组（记录从根到该试卷的所有分类名），方便 UI 进行分组和面包屑显示。
